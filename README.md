# Kamehameha.js

Bring legendary saiyans to your console.

## How to use

```js
var Kamehameha = require('kamehameha');

Kamehameha.release({ images: ["logo"], delay: 500 }, function() {
    // Charge your ki for the next one.
});
```


## Available characters

- puar
- logo
- vegeta
- goku
- krillin
- piccolo
- gohan
